Dragon Storm

Play Game

Dragon Storm is an interactive game in which users fly a dragon using key board movements in attempt to burn down structures for as long as they can.

    Dragon Storm was programmed using:
  • Javascript (ES6)
  • HTML5
  • CSS
  • Webpack

I integrated the Canvas API, using JavaScript, to draw graphics, animations, and photo-manipulation in real-time to create a dynamic experience for users. Event-listeners were also implemented in order to trigger fire blowing effects and dragon movements on ‘keyup’ and ‘keydown’ to create responsive and dynamic game play. I programmed asynchronous setTimeOut and setInterval to trigger various phases of the game-logic. Lastly, I used DOM manipulation to trigger the rendering of key-presses and clicks using a combination of CSS and JavaScript’s classList.

I designed all of the graphic art/ designs using Procreate and ADOBE Illustrator, including all graphic art, designs, colors, and layout.
