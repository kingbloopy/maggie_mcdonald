Plantsy

Live Site

Plantsy is a full-stack clone of Esty where users can create/sell products, leave reviews, and add products to their cart. / Github

    Plantsy was programmed using:
  • ReactJS
  • Ruby on Rails
  • PostgreSQL
  • CSS
  • SASS
  • HTML5
  • Javascript
  • jQuery
  • Webpack
  • Git
  • Heroku

I utilized CSS media queries and columns, creating the classic masonry style to aesthetically render a products index. In addition, mapStateToProps were used to dispatch thunk actions, normalizing the state shape, and efficiently updating elements on the page. I efficiently designed active record queries to fetch posts by product category and effectively utilized a single component using a switch operator to DRY up code and display the fetched category. Pop-up modals were dynamically streamlined using thunk actions and a switch operator that takes in a form-type parameter to create a seamless, single-page, user experience. I utilized Amazon Web Services from which to pull photos as well as PostgreSQL to store and pull data. Plantsy was also built using a Ruby on Rails backend with Active Record models and controllers as well as Jbuilder to transpile rails data into JSON for a ReactJS frontend.

I designed the majority of the graphic art/ designs using Procreate and ADOBE Illustrator, such as the logos and color schemes.
